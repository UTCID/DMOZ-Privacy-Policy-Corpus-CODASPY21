This code is executed on our server or on your device.What is a webbeacon?A web beacon (or a pixel tag) is a small, invisible piece of text or image on a website that is used to monitor traffic on a website. In order to do this, various data about you is stored using web beacons.ConsentWhen you visit our website for the first time, we will show you a pop-up with an explanation about cookies.
