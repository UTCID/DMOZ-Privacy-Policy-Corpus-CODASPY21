* Do you know what the CAA record is?
Suggested article: What is CAA record?
The CAA record or Certification Authority Authorization is one of the resource DNS records. It has the job to set all the sides that can create certificates for a particular domain. It could be one or more.
How to add a CAA record?
Inside the CAA record, you will see:
Flag – sets if somebody can or can’t issue certificates for the domain
Type – type of certificate: issue, issuewild, iodef
Value – the domain name of the Certification Authority
Setting a CAA record policy will cover the whole domain, including the subdomains. If you set it up for example.com, then it will cover subdomain.example.com.
