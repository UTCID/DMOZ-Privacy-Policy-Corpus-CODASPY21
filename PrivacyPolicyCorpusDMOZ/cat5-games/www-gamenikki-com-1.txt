* Do you know what the TLSA record is?
Suggested article from our blog: What is TLSA record
The TLSA record or Transport Layer Security Authentication record is used to associate a TLS server certificate or public key with the domain name where the record is found.
TLSA record has the following components:
Port number
Protocol
Hostname
It also has the following data elements:
Certificate usage: A numeric value (0-255).
Selector: A numeric value (0-255).
Matching type: A numeric value (0-255).
Certificate association data: Hexadecimal.
Remember that the TLSA record can only be trusted if DNSSEC is enabled on your domain. If you are interested in TLSA records, you can find more information in this article!
