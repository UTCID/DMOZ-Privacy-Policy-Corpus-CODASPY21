COOKIE POLICY
In practical and non-technical terms, a cookie is similar to a small file stored on your computer by websites while browsing, useful to save your preferences and improve the performance of web sites.
In this way the browsing experience of the user is optimized.
In detail, a cookie is a small size text string sent by a web server to a web client (usually a browser) and then sent back by the client to the server (without change) each time the client logs on to the same portion of the same web domain.
(Source:
https://en.wikipedia.org/wiki/HTTP_cookie)
