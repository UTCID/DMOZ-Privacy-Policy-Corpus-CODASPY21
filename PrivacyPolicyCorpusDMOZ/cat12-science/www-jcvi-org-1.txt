1. Collection of Personal Information
JCVI does not automatically collect any personal information about user when user visits Sites. Personal information is only collected from user if user provided that information voluntarily. Examples of voluntarily provided information include sending JCVI an e-mail or completing an online form. JCVI will use the information provided by user to respond to user's message or inquiry. E-mails are treated in the same manner as letters sent to JCVI, therefore, information will not be collected for commercial marketing.
2. Information Automatically Collected
Although JCVI does not collect any personal information, there is some information that is automatically collected. The information that is automatically collected includes:
1. The Internet Protocol (IP) address or host name of the computer that accessed the website. An IP address only identifies the computer and not the user.
2. The type of browser, its version and the operating system used to access the Sites.
3. The date and time user accessed the Sites.
4. The pages user visits.
5. The content length of any document sent to user.
6. The request method used.
7. The Universal Resource Identifier (URI) and any associated query string which defines the location of a resource on the server.
8. The transport protocol and the version used.
9. If user linked to the Sites from another website, the address of that website.
The information above is automatically collected to help make Sites more useful to its users. JCVI usually does not create a record or track information about individual users and their visits. However, JCVI reserves the right to track users if JCVI reasonably believes that the Sites are being wrongfully accessed and/or abused.
3. Cookies
JCVI may use cookies. A cookie is a small file containing information that is sent to a user's Web browser to distinguish users of web sites. The use of cookies is a regular practice among Internet web sites. Sites may occasionally use both persistent and non-persistent cookies to enhance or improve user's visit to Sites. Non-persistent cookies are created automatically on the instrument user is employing to access Sites. A non-persistent cookie is erased when user's Web browser is closed. Persistent cookies are stored on user's hard drive. These types of cookies allow Sites to recognize user upon subsequent visits which may facilitate user's preference for certain features available on Sites. User should note that the majority of Web browsers accept cookies, however there may be options available on user's browser that may prevent the accepting or receiving of cookies. If cookies are not accepted, there is a possibility that the complete functionality of Sites may be impaired.
